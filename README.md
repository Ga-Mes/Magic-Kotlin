Магический плагин:
были добавлены кастомные предметы, которые дают носителю особенные способности.

Установка:
положить файл Magic-1.0.0.jar в папку plugins сервера, запустить сервер.

Предметы:
- Амулет защиты (уменьшает урон наносимый вам)
- Амулет силы (увеличивает урон наносимый вами)
- Амулет прыжка (увеличивает прыжок)

Предметы можно получить в выживании в структурах:
- Амулет защиты в джунглевом храме (30% вероятность)
- Амулет силы в особняке тёмного леса (40% вероятность)
- Амулет прыжка в закопанном сокровище (50% вероятность)

Доступные комманды:
- /playerdata save (сохраняет данные о игроке в файле playerdata.json в папке плагина, доступна для всех)
- customgive <id> (выдаёт кастомный предмет согласно id (начиная с 0 и заканчивая 2), доступна для операторов)

Структура файла сохранения данных игроков:
1. Текущее здоровье
2. Список кастомных предметов в наличии
3. Продолжительность времени прошедшего от захода на сервер до вызова комманды /playerdata save в минутах
